# TradingGym Legal Website

This repository hosts the public legal pages for the TradingGym iOS app.

## Included Pages

- `index.html`: Privacy Policy
- `terms.html`: Terms of Service
- `support.html`: Support
- `styles.css`: Shared site styling

## GitHub Pages Setup

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to **Settings** -> **Pages**.
4. Under **Build and deployment**, select:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` and `/ (root)`
5. Save changes and wait for deployment.

## Site URL Format

Your published site URL will be:

`https://<github-username>.github.io/<repository-name>/`

If this repository is named `TradingGym-legal`, the URL will look like:

`https://<github-username>.github.io/TradingGym-legal/`
